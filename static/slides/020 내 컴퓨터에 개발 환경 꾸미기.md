# 내 컴퓨터에 개발 환경 꾸미기 (1)

### .NET Core, Node.js 및 도구 설치

---

## 오늘의 목표

HandStack 프레임워크를 사용하기 위해 가장 먼저 해야 할 일!

- 내 컴퓨터에 HandStack 실행에 <mark>필수적인 프로그램</mark>들을 설치합니다.
- 각 프로그램이 <mark>왜 필요한지</mark> 간단하게 알아봅니다.
- 설치가 <mark>잘 되었는지 확인</mark>하는 방법을 배웁니다.

---

## HandStack의 엔진을 설치해요

> 자동차에 엔진이 있어야 움직이듯,
> HandStack도 <mark>.NET Core</mark>와 <mark>Node.js</mark>라는 엔진이 꼭 필요해요!

- HandStack의 핵심 기능들이 이 두 가지를 기반으로 만들어졌기 때문입니다.
- 지금부터 이 두 개의 엔진을 설치해 보겠습니다.

---

## 첫 번째 엔진: .NET Core 설치

### .NET Core는 왜 필요한가요?

- HandStack의 서버(백엔드)와 CLI(명령어 도구)가 .NET Core 기술로 만들어졌습니다.
- 안정적이고 빠른 서버를 만드는 데 사용되는 마이크로소프트의 오픈소스 기술입니다.

### 설치 방법

- .NET 공식 웹사이트에 접속하여 <mark>LTS(Long Term Support)</mark> 버전을 다운로드하여 설치합니다.
- LTS 버전은 오랫동안 안정적으로 지원되는 버전이라 추천합니다.

---

## .NET Core 설치 확인하기

설치가 완료되면, 터미널이나 명령 프롬프트 창을 열고 아래 명령어를 입력해 보세요.

```bash
dotnet --version
```

- 위와 같이 설치된 버전 번호가 나타나면 성공입니다.
- 예시: `8.0.300`

> 오류가 발생하면 컴퓨터를 재시작하거나 설치 과정을 다시 확인해 보세요.

---

## 두 번째 엔진: Node.js 설치

### Node.js는 왜 필요한가요?

- HandStack의 서버(백엔드)와 CLI(명령어 도구)가 Node.js 기술로 만들어졌습니다.
- 빠르고 효율적인 웹 서비스를 만들 때 널리 사용되는 기술입니다.

### 설치 방법

- Node.js 공식 웹사이트에 접속하여 <mark>LTS(Long Term Support)</mark> 버전을 다운로드하여 설치합니다.
- .NET Core와 마찬가지로 안정적인 LTS 버전을 선택합니다.

---

## Node.js 설치 확인하기

터미널이나 명령 프롬프트에서 아래 두 명령어를 차례대로 입력합니다.

```bash
node -v
```

```bash
npm -v
```

- `node -v`는 Node.js 버전을, `npm -v`는 npm(패키지 관리자) 버전을 보여줍니다.
- 버전 번호가 보이면 성공적으로 설치된 것입니다.

---

## 잠깐! `nuget`과 `npm`이 뭔가요?

두 가지 모두 <mark>패키지 관리자</mark>입니다.

- **패키지 관리자란?**
    - 개발에 필요한 다양한 도구(패키지, 라이브러리)들을 쉽게 설치하고 관리해주는 프로그램입니다.
    - 마치 스마트폰의 앱 스토어와 비슷한 역할을 합니다.

- `nuget` : .NET 개발자들이 사용하는 패키지 관리자입니다.
- `npm` : Node.js 개발자들이 사용하는 패키지 관리자입니다.

HandStack은 이 두 가지를 모두 활용하여 개발 생산성을 높입니다.

## HandStack CLI 및 Visual Studio 2022 Community 설치

GitHub 에서 HandStack 개발의 <mark>핵심 도구</mark>들을 설치하고 설정합니다.

- 개발 비서, <mark>HandStack CLI</mark> 설치하기
- 코딩 놀이터, <mark>Visual Studio 2022 Community</mark> 설치하기
- 개발을 더 편하게 만들어 줄 <mark>추천 확장 프로그램</mark> 설정하기

---

## 여러분의 개발 비서, HandStack CLI

> <mark>HandStack CLI</mark>는 여러분의 개발 비서 같은 역할을 해요.
> 프로젝트 생성, 모듈 관리, 빌드, 배포 등 복잡하고 반복적인 작업을
> <mark>명령어 한 줄</mark>로 간단하게 처리해 줍니다.

![center](httpsa://www.handstack.io/img/logo.svg)

---

## HandStack CLI 설치하기

터미널 또는 명령 프롬프트를 열고 아래 명령어를 순서대로 입력하세요.
(Git이 설치되어 있어야 합니다.)

1. HandStack 소스 코드 복제하기
   ```bash
   git clone https://github.com/handstack77/handstack.git
   ```

2. 생성된 디렉토리로 이동하기
   ```bash
   cd handstack
   ```

3. 설치 스크립트 실행하기
   - Windows: `install.bat`
   - macOS/Linux: `sh install.sh`

---

## HandStack CLI 설치 확인

설치가 잘 되었는지 확인해 볼까요?
설치 스크립트 실행이 완료된 후, 새 터미널 창을 열고 아래 명령어를 입력해 보세요.

```bash
ack
```

- HandStack CLI(`ack` 프로그램)의 사용법 안내가 나타나면 성공입니다.
- `ack: command not found`와 같은 오류가 발생하면, 터미널을 완전히 껐다 다시 켜거나 컴퓨터를 재시작한 후 시도해 보세요.

---

## 코딩 놀이터, Visual Studio 2022 설치

HandStack으로 만든 코드를 보고 수정하려면 <mark>코드 편집기</mark>가 필요합니다.
Visual Studio 2022 Community는 강력한 기능을 갖춘 <mark>무료</mark> 코드 편집기입니다.

- 공식 웹사이트에서 `Community` 버전을 다운로드하여 설치합니다.
- 설치 시 워크로드 선택 화면에서 <mark>ASP.NET 및 웹 개발</mark>을 꼭 체크해 주세요.

---

## VS 2022 추천 확장 프로그램 설치

Visual Studio를 더 강력하게 만들어주는 추가 기능들입니다.
Visual Studio 상단 메뉴의 `확장` > `확장 관리`에서 아래 이름으로 검색하여 설치하세요.

- `Bundler & Minifier 2022+`
- `Microsoft Library Manager`
- `Open Command Line`
- `Productivity Power Tools 2022`

---

## 나만의 코딩 환경 만들기

개발을 편안하게 하려면 개인 설정이 중요합니다.

- **테마 변경**: `도구` > `테마`에서 어둡게/밝게 등 원하는 테마를 선택할 수 있습니다.
- **글꼴 변경**: `도구` > `옵션` > `환경` > `글꼴 및 색`에서 코딩용 글꼴(D2Coding 추천)과 크기를 조절할 수 있습니다.

자신에게 가장 편안한 환경을 만들어 보세요.

---

## Windows 사용자 꿀팁: Winget

Windows 사용자라면 <mark>Winget</mark>을 사용해 보세요.
명령어 한 줄로 다양한 프로그램을 쉽게 설치할 수 있는 패키지 관리자입니다.

- 예를 들어, Visual Studio Code를 설치하고 싶다면?
  ```bash
  winget install Microsoft.VisualStudioCode
  ```

- HandStack 개발에 유용한 다른 도구들도 Winget으로 간편하게 설치할 수 있습니다.
