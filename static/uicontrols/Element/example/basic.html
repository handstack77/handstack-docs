Element는 전용 컨트롤이 없는 일반 엘리먼트(div, span 등)를 위한 범용 컨트롤입니다. syn-options의 content 옵션에 따라 값을 어디에 읽고 쓰는지가 달라집니다.

1) content: 지정 안 함 (기본값, innerText 사용)
기본 텍스트
2) content: 'content' (textContent 사용)
textContent 값
3) content: 'html' (innerHTML 사용, HTML 태그 포함 가능)
굵게 표시된 텍스트
4) content: 'content' + dataType: 'number' (숫자 형식 변환)
0
5) content: 'content' + dataType: 'bool' (checkedValue/uncheckedValue 로 치환)
N
