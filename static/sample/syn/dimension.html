속성

$dimension version을 반환합니다.

메서드

화면 내 최대 표시 크기를 조회합니다.
브라우저의 최대 표시 크기를 조회합니다.
지정된 노드 또는 화면 내 스크롤 거리를 조회합니다.
지정된 노드를 선택하는 마우스의 위치를 조회합니다.
화면 내에서 지정된 노드의 위치를 조회합니다.
지정된 노드에서 최상위 노드 간의 left 거리 합계를 조회합니다.
지정된 노드의 부모 노드 간의 left 거리를 조회합니다.
지정된 노드에서 최상위 노드 간의 top 거리 합계를 조회합니다.
지정된 노드의 부모 노드 간의 top 거리를 조회합니다.
지정된 노드의 크기를 조회합니다.
지정된 텍스트 문자열의 width 크기를 조회합니다. fontSize에 따라 달라집니다.
지정된 텍스트 문자열의 height 크기를 조회합니다. fontSize에 따라 달라집니다.
