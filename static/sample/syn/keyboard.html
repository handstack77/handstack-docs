syn.$k: 키보드 단축키 기능을 제공합니다.

속성

키 이름(backspace, enter, a, f1 등)과 keyCode 숫자값의 매핑 테이블(읽기 전용)입니다.
KeyboardEvent.code 값(Backspace, KeyA, F1 등)과 keyCodes 키 이름의 매핑 테이블(읽기 전용)입니다.
setElement()로 마지막으로 지정된 대상 노드의 id 입니다. 아래 setElement() 실행 후 갱신됩니다.
setElement()로 등록된 노드별 keydown/keyup 콜백 목록입니다. 아래 setElement() 실행 후 갱신됩니다.

메서드

사용자의 키보드 단축키 정보를 수신할 노드를 설정합니다.
단축키에 대응하는 keyType, keyCode, function 을 설정합니다.
단축키에 대응하는 keyType, keyCode 을 삭제합니다.
KeyboardEvent.code 값(예: 'KeyA')에 대응하는 keyCode 숫자값을 조회합니다.
