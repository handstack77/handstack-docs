syn.$r: http 요청 및 URL, Cookie 조작 기능 제공합니다.

속성

현재 url 주소에서 조회한 queryString 값을 key/value 형태로 보관합니다.
현재 페이지의 location.pathname 값을 반환합니다.

메서드

$r.params에서 키 값을 조회합니다.
현재 url에서 $r.params 데이터로 queryString을 생성합니다.
Json 객체의 데이터를 queryString url로 변환합니다.
url 주소의 queryString 데이터를 Json 객체로 변환합니다.
상대 경로를 기준 url(baseUrl 또는 현재 페이지 주소)에 대한 절대 경로 url로 변환합니다.
url에 queryString 파라미터를 추가합니다.
url에서 queryString 파라미터를 제거합니다.
url의 queryString 파라미터 값을 설정(대체)합니다.
url 주소의 컨텐츠에 접근 가능 여부를 확인합니다.
fetch API 기반으로 http 요청을 수행합니다. (Proxy 객체의 send 액션만 지원)
