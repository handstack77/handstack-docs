$array: Array 객체에 대한 확장 함수를 제공합니다.

속성

syn.js Module 프레임워크의 버전 문자열을 반환합니다($array 등 개별 모듈에는 자체 version 속성이 없습니다).

메서드

배열의 항목에서 중복된 항목을 삭제합니다.
배열의 항목에서 정순, 역순 정렬후 배열을 반환합니다.
객체 배열의 항목에서 정순, 역순 정렬후 배열을 반환합니다.
객체 배열의 항목에서 그룹화후 배열을 반환합니다.
배열의 항목을 무작위로 섞은 후 배열을 반환합니다.
객체 배열의 항목을 지정된 위치에 추가 후 배열을 반환합니다.
지정된 위치에 있는 객체 배열의 항목을 삭제 후 배열을 반환합니다.
배열에 지정된 값이 포함되어 있는지 확인 후 boolean 값을 반환합니다.
predicate 조건으로 중복 여부를 판단하여 두 배열을 병합한 배열을 반환합니다(기본 predicate: a === b).
두 배열의 합집합(중복 제거)을 반환합니다.
sourceArray에서 targetArray에 없는 항목만 남긴 차집합을 반환합니다.
두 배열에 공통으로 존재하는 항목의 교집합을 반환합니다.
두 배열 중 한쪽에만 존재하는 항목(대칭차집합)을 반환합니다.
파라미터 형태(ParameterName/Value 등)의 객체 배열에서 이름으로 값을 찾아 반환하며, 없으면 기본값을 반환합니다.
배열의 항목 데이터에 순위(동점 처리 포함)를 적용한 배열을 반환합니다.
구분자로 분리된 문자열을 트리밍 후 빈 항목을 제거한 배열로 변환합니다.
