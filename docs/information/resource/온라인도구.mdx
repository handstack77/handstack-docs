# 온라인 도구 Online Tools

## 개발 도구

[URL-encoded](https://www.urlencoder.org/), [URL-decoded](https://www.urldecoder.org/) - URL 인코딩 형식으로 인코딩/디코딩하기
> URL 인코딩 형식을 처리해야 할 경우 간단한 온라인 도구를 사용해 데이터를 인코딩하거나 디코딩 서비스를 제공합니다.

[Instant SQL Formatter & Beautifier](https://www.dpriver.com/pp/sqlformat.htm) - SQL 문을 읽기 쉬운 형식으로 자동으로 포맷팅해주는 도구
> MS Access, DB2, MSSQL, MySQL, Oracle 등 데이터베이스를 지원하며, 출력 결과를 SQL, HTML, C#, Javas, XML 등 다양한 형식으로 지원합니다.

[ConvertCSV](https://www.convertcsv.com/) - CSV 파일을 변환하고 다양한 형식으로 변환하는 도구
> CSV 파일을 변환하고 다양한 형식으로 변환하는 도구로, CSV 파일을 JSON, XML, SQL, Excel, HTML, Markdown 등 다양한 형식으로 변환할 수 있습니다.

[JSON Editor Online](https://jsoneditoronline.org/) - JSON 데이터를 생성, 편집, 검증할 수 있는 도구
> 사용자가 복잡한 JSON 구조를 쉽게 시각화하고 생성, 수정할 수 있도록 도와주며, 다양한 스키마 형식에 대해 JSON 데이터를 검증할 수 있습니다.

[Table to Markdown](https://markdown-convert.com/en/tool/table) - 마크다운 테이블을 생성하는 도구
> 다양한 표를 마크다운 표기법으로 변환하는 도구로, Excel, CSV, JSON, SQL 결과 및 기타 표 형식에서 마크다운 표를 생성합니다.

[HTML Beautifier & Formatter](https://www.cleancss.com/html-beautify/), [CSS Formatter](https://www.cleancss.com/css-beautify/), [Javascript Formatter](https://www.cleancss.com/javascript-beautify/) - HTML, CSS, Javascript 코드를 읽기 쉬운 형식으로 자동으로 포맷팅해주는 도구

[Javascript Minifier](https://www.cleancss.com/javascript-minify/), [Javascript Obfuscate](https://www.cleancss.com/javascript-obfuscate/), [CSS Minifier](https://www.cleancss.com/css-minify/), [HTML Minifier](https://www.cleancss.com/html-minify/) - HTML, CSS, Javascript 코드를 압축 및 난독화 하는 도구

[Convert HTML to Markdown](https://www.convertsimple.com/convert-html-to-markdown/) - HTML을 마크다운으로 변환하는 도구
> HTML을 붙여넣거나 업로드하여 마크다운으로 변환하는 도구입니다.

[BeautifyTools](https://beautifytools.com/) - HTML, CSS, Javascript 코드를 읽기 쉬운 형식으로 자동으로 포맷팅해주는 도구

[Clipboard 2 Markdown](https://euangoddard.github.io/clipboard2markdown/) - 클립보드에 복사한 텍스트를 마크다운으로 변환하는 도구
> 클립보드에 복사한 텍스트를 마크다운으로 변환하는 도구로, 텍스트를 붙여넣기하면 마크다운으로 변환된 텍스트를 얻을 수 있습니다.

[Tables Generator](https://www.tablesgenerator.com/markdown_tables) - 마크다운 표 생성기

[onlinepngtools](https://onlinepngtools.com/create-transparent-png) - 투명한 PNG 이미지를 생성하는 도구
> 투명한 PNG 이미지를 생성하는 도구로, 이미지를 업로드하거나 URL을 입력하여 투명한 PNG 이미지를 생성할 수 있습니다.

[htmlcolorcodes](https://htmlcolorcodes.com/) - HTML 색상 코드를 생성하는 도구
> HTML 색상 코드를 생성하는 도구로, 색상 코드를 선택하거나 입력하여 색상 코드를 생성할 수 있습니다.

[json2yaml](https://www.json2yaml.com/) - JSON을 YAML로 변환하는 도구
> YAML은 복잡한 데이터를 간결하고 읽기 쉬운 형식으로 저장하는 데 사용됩니다. 주로 설정 파일의 형식으로 사용됩니다.

[regex101](https://regex101.com/) - 정규식을 테스트하고 설명하는 도구
> 정규식을 테스트하고 설명하는 도구로, 정규식을 입력하고 테스트할 텍스트를 입력하면 정규식이 어떻게 작동하는지 확인할 수 있습니다.

[realfavicongenerator](https://realfavicongenerator.net/) - Favicon 생성기
> 이미지를 업로드하여 Favicon을 생성하는 도구로, Favicon을 생성하고 다양한 플랫폼에 대응하는 Favicon을 생성할 수 있습니다.

[json-to-jsonschema](https://jsonformatter.org/json-to-jsonschema) - JSON을 JSON 스키마로 변환하는 도구
> JSON을 JSON 스키마로 변환하는 도구로, JSON을 붙여넣으면 JSON 스키마로 변환된 결과를 얻을 수 있습니다.

[quicktype](https://app.quicktype.io/) - JSON 데이터를 분석하여 코드를 생성하는 도구
> JSON 데이터를 분석하여 C#, C++, Dart, Go, Java, Javascript, Kotlin 등등 코드를 생성하는 도구로, JSON 데이터를 붙여넣으면 다양한 언어의 코드를 생성할 수 있습니다.

[shaper.design](https://shaper.design) - UI 인터페이스용 CSS 디자인 시스템 생성 도구
단 몇 초 만에 다양한 CSS 디자인 레이아웃 시스템을 살펴보세요. 디자인 토큰을 정의하고 실시간으로 변경 사항을 시각화 도구를 제공합니다. 결과물은 단순한 그림이 아니라 다른 디자인 도구에서 사용할 수 있는 스타일 세트와 프로젝트에 복사할 수 있는 코드입니다.

[shuffle](https://shuffle.dev/) - CSS 그리드 레이아웃을 생성하는 도구
Tailwind CSS, Bootstrap, Bulma, Foundation, Materialize, Semantic UI, UIKit 등의 CSS 프레임워크를 지원하며, CSS 그리드 레이아웃을 생성할 수 있습니다.
