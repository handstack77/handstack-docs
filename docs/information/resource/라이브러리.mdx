# 라이브러리 Library

## Javascript

[SQL Formatter](https://github.com/sql-formatter-org/sql-formatter) - SQL 쿼리를 친숙하게 포맷팅을 하기 위한 JavaScript 라이브러리
> GCP BigQuery, IBM DB2, Apache Hive, MariaDB, MySQL, Couchbase N1QL, Oracle PL/SQL, PostgreSQL, Amazon Redshift, SingleStoreDB, Snowflake, Spark, SQL Server Transact-SQL, Trino(및 Presto) 등 다양한 SQL 포맷팅을 지원

[Squel.js](https://github.com/hiddentao/squel) - SQL 쿼리를 생성하기 위한 JavaScript 라이브러리
> 유연하고 강력한 Javascript용 SQL 쿼리 문자열 빌더입니다. Squel은 더 이상 적극적으로 유지 관리되지 않습니다. 가끔 버그 수정과 소규모 작업에만 시간이 있습니다.
