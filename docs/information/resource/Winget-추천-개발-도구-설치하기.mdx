# Winget 추천 개발 도구 설치하기

소프트웨어 개발자와 운영자를 위한 좋은 도구는 생산성을 10배 이상 높일 수 있습니다. 이 글에서는 Windows에서 Winget을 사용하여 개발자와 운영자를 위한 추천 도구를 설치하는 방법을 소개합니다. Windows 11+ 에서는 기본으로 적용되어 있습니다.

Windows 10+ 에서 Winget을 설치하는 방법은 다음과 같습니다:

1. **Microsoft Store에서 설치하기**:
    - [앱 설치 관리자 설치하기](https://apps.microsoft.com/detail/9nblggh4nns1?rtc=1&hl=ko-kr&gl=KR#activetab=pivot:overviewtab)
    - 앱 설치 관리자 페이지에 앱을 설치하고 설치가 완료되기를 기다립니다.

2. **GitHub에서 설치하기**:
    - 웹 브라우저로 https://aka.ms/getwinget 설치 관리자를 다운로드 합니다.
    - 다운로드 파일을 두 번 클릭하여 Winget 설치 프로그램을 시작합니다.

설치가 완료되면 `winget` 명령을 사용할 수 있습니다.

## IDE 및 확장 프로그램

Visual Studio 2022 Community - .NET 개발을 하실건가요? 고민하지 마세요. Visual Studio로 시작하세요. Visual Studio 2022 Community는 무료로 사용할 수 있는 최고의 IDE입니다.

```bash
winget install -e --id=Microsoft.VisualStudio.2022.Community
```

Visual Studio Code - Windows, macOS, Linux에서 동일한 개발 경험을 제공하는 가벼운 텍스트 에디터에서 강력한 IDE로 발전한 Visual Studio Code는 다양한 언어와 확장 프로그램을 지원합니다.

```bash
winget install -e --id=Microsoft.VisualStudioCode
```

Notepad++ - 내가 아는 최고의 기본 텍스트 편집기 입니다. Notepad++은 다양한 언어를 지원하고, 플러그인을 통해 다양한 기능을 확장할 수 있습니다.

```bash
winget install -e --id=Notepad++.Notepad++
```

WinMerge - 파일 및 디렉토리 비교 및 병합 도구로, 파일 및 디렉토리의 차이점을 비교하고, 병합할 수 있습니다.

```bash
winget install -e --id=WinMerge.WinMerge
```

DBeaver - DBeaver는 데이터베이스 관리 도구로, MySQL, PostgreSQL, SQLite, Oracle, SQL Server 외에도 다양한 데이터베이스를 지원하며, SQL 쿼리를 실행하고, 데이터베이스를 관리할 수 있습니다.

```bash
winget install -e --id=dbeaver.dbeaver
```

DB Browser for SQLite - SQLite 데이터베이스를 관리하고, 테이블을 생성하고, 데이터를 삽입하고, 쿼리를 실행할 수 있는 무료 도구입니다.

```bash
winget install -e --id=DBBrowserForSQLite.DBBrowserForSQLite
```

## Windows 유틸리티

Free Commander XE - Windows 탐색기 보다 더 나은 대안 중 하나입니다. 듀얼 디스플레이, 탭, 즐겨찾는 디렉토리, 필터링 등을 제공합니다.

```bash
winget install -e --id=MarekJasinski.FreeCommanderXE
```

Everything - 디스크 드라이브를 인덱싱하고 와일드카드 및 Regex를 지원하여 즉각적인 파일 검색 기능과 기본 탐색기와 비교할 수 없는 빠른 속도를 제공합니다.

```bash
winget install -e --id=voidtools.Everything
```

DeepL - 빠르고 정확하며 고품질 번역을 위한 최고의 번역 도구입니다. 수백만 명의 사람들이 매일 26개 언어를 번역하고, 의사소통을 개선하고, 언어 장벽을 허무는 데 이 앱을 사용합니다.

```bash
winget install -e --id=DeepL.DeepL
```

WinSCP - 오픈소스 기반의 무료 FTP, SFTP, WebDAV, S3, SCP 클라이언트입니다. 주요 기능은 로컬 컴퓨터와 원격 컴퓨터 간의 파일 전송입니다. 이 외에도 기본 파일 관리자 기능을 제공합니다.

```bash
winget install -e --id=WinSCP.WinSCP
```

FortiClient VPN - Fortinet의 보안 솔루션을 사용하는 사용자를 위한 VPN 클라이언트입니다. FortiClient는 다양한 보안 기능을 제공하며, 사용자의 보안을 강화합니다.

```bash
winget install -e --id=Fortinet.FortiClientVPN
```

Postman - Postman은 API 개발을 위한 협업 플랫폼으로, API를 설계, 개발, 테스트, 문서화하는 데 사용할 수 있습니다.

```bash
winget install -e --id=Postman.Postman
```

IrfanView - 현존하는 최고의 이미지 뷰어 중 하나입니다. 빠르고, 훌륭한 UX를 갖추고 있으며, 수많은 유용한 기능을 갖추고 있습니다.

```bash
winget install -e --id=IrfanSkiljan.IrfanView
```

Process Explorer - Windows의 작업 관리자의 확장판입니다. 실행 중인 프로세스를 표시하는 동일한 기능 외에도 덤프 파일을 생성하고, 성능 카운터를 표시하고, 핸들로 프로세스를 찾는 등의 작업을 수행할 수 있습니다.

```bash
winget install -e --id=Microsoft.Sysinternals.ProcessExplorer
```

Chrome 브라우저 - 구글에서 제공하는 빠르고 안정적인 웹 브라우저입니다. Chrome은 다양한 확장 프로그램을 지원하며, 사용자의 웹 브라우징 경험을 향상시킵니다.

```bash
winget install -e --id=Google.Chrome
```

Brave 브라우저 - 개인 정보 보호를 중요시하는 사용자를 위한 빠르고 안전한 웹 브라우저입니다. Brave은 광고 차단, 추적 방지, HTTPS 업그레이드 등 다양한 보안 기능을 제공합니다.

```bash
winget install -e --id=Brave.Brave
```

Mattermost - 오픈 소스 팀 채팅 및 협업 도구로, 팀원들과 채팅하고, 파일을 공유하고, 작업을 추적하고, 통합된 알림을 받을 수 있습니다.

```bash
winget install -e --id=Mattermost.MattermostDesktop
```

Zoom - 비디오 회의, 웨비나 등 모든 커뮤니케이션 요구 사항을 충족하는 안정적인 비디오 플랫폼을 제공합니다.

```bash
winget install -e --id=Zoom.Zoom
```

X-Mouse Button Control - 마우스 버튼을 사용자 정의하고, 다양한 기능을 추가할 수 있는 무료 마우스 유틸리티입니다.

```bash
winget install -e --id=Highresolution.X-MouseButtonControl
```

반디집 - 빠른 처리 속도와 편리한 기능을 제공하는 강력한 압축 프로그램입니다. 다양한 압축 포맷을 지원하며, 암호화, 분할, 병합, 검색, 검증 등 다양한 기능을 제공합니다.

```bash
winget install -e --id=Bandisoft.Bandizip
```

카카오톡 - 국내에서 인기있는 무료로 사용할 수 있는 메시징 앱으로, 텍스트, 음성, 영상 메시지를 보낼 수 있으며, 그룹 채팅, 브로드캐스트, 스티커, 이모티콘 등 다양한 기능을 제공합니다.

```bash
winget install -e --id=Kakao.KakaoTalk
```

foobar2000 - 사용자 정의 가능한 인터페이스, 고품질 MP3, MP4, AAC, CD 오디오 재생을 지원하는 가볍고 빠른 무료 음악 플레이어입니다.

```bash
winget install -e --id=PeterPawlowski.foobar2000
```

PicPick - 화면 캡처, 이미지 편집, 색상 선택, 화면 측정 등 다양한 기능을 제공하는 무료 이미지 편집 도구입니다.

```bash
winget install -e --id=NGWIN.PicPick
```

## 소스 제어(git, svn) 관리

Git - Git은 분산 버전 관리 시스템으로, 소스 코드를 효과적으로 관리할 수 있습니다. Git은 빠르고 확장 가능하며, 오픈 소스이기 때문에 무료로 사용할 수 있습니다.

```bash
winget install -e --id=Git.Git
```

Git Extensions - Git용 오픈 소스 UI 클라이언트입니다. Git Extensions는 Windows에서 사용하기 쉽고, Git의 기능을 확장하여 사용할 수 있습니다.

```bash
winget install -e --id=GitExtensionsTeam.GitExtensions
```

TortoiseSVN - Subversion을 위한 Windows용 도구입니다. Windows 탐색기에 통합되어 있어, 파일 및 디렉토리의 상태를 쉽게 확인하고, Subversion 명령을 실행할 수 있습니다.

```bash
winget install -e --id=TortoiseSVN.TortoiseSVN
```

```bash
winget install -e --id=Git.Git
```

Beyond Compare - 파일이나 디렉터리의 차이점을 나란히 보여주는 비교 도구입니다. 파일 비교, 디렉토리 비교, 병합, 동기화 등 다양한 기능을 제공합니다.

```bash
winget install -e --id=ScooterSoftware.BeyondCompare4
```

## 개발 환경 SW

### Winget 을 이용한 .NET Core 설치

.NET 8.0 - 마이크로소프트에서 제공하는 C# 언어로 개발 하는 크로스 플랫폼, 고성능, 오픈 소스 프레임워크입니다.

```bash
winget install -e --id=Microsoft.DotNet.SDK.8
```

### Winget 을 이용한 Node.js 설치

Node.js - 오픈 소스이며 크로스 플랫폼 JavaScript 런타임 환경으로, 거의 모든 종류의 프로젝트에 인기 있는 도구입니다.

```bash
winget install -e --id=OpenJS.NodeJS.LTS
```

## 분석 및 디버깅 도구

LINQPad - LINQPad은 C#과 .NET 언어를 사용하여 데이터베이스 쿼리를 실행하고, LINQ 쿼리를 작성하고, C# 코드를 실행할 수 있는 인터랙티브 쿼리 도구입니다.

```bash
winget install LINQPad.LINQPad.7
```

Fiddler Everywhere - HTTP 디버깅 프록시 도구로, HTTP 요청을 가로채고 수정하고, 응답을 확인하고, 성능을 분석할 수 있습니다.

```bash
winget install -e --id=Telerik.Fiddler.Everywhere
```

PerfView - 마이크로소프트에서 제공하는 .NET용 무료 프로파일러입니다. PerfView는 .NET 프로그램의 성능 문제를 진단하고 해결하는 데 도움이 됩니다.

```bash
winget install -e --id=Microsoft.PerfView
```

dnSpy - ILSpy를 기반으로 한 디컴파일러이지만 디버거이기도 합니다. 실행 중인 .NET 프로세스에 연결하고 해당 프로세스에 대한 소스 코드나 기호가 없더라도 디버깅할 수 있습니다.

```bash
winget install -e --id=dnSpyEx.dnSpy
```
