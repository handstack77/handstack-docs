# 5kb 미만 Javascript 오픈소스 목록

MicroJS 웹 사이트는 작고 강력한 마이크로 프레임워크와 마이크로 라이브러리를 발견하고 선택할 수 있도록 도와줍니다. 이 사이트는 종속성이 없고, 압축된 상태에서 5KB 이하인 라이브러리만을 다룹니다.

그래서 간결하고 효율적인 도구를 제공하여, 대규모 프레임워크 대신 작고 특정한 기능을 잘 수행하는 라이브러리를 사용할 수 있게 합니다. 예를 들어, DOM 조작, 이벤트 처리, 애니메이션 등 다양한 작업을 수행할 수 있는 라이브러리를 찾을 수 있습니다.

HandStack 에서도 영감을 받아 도입하거나 활용해서 프로젝트를 진행하고 있어, 목록을 소개합니다. 향후에도 새로운 라이브러리와 프레임워크를 추가하고 싶다면 GitHub 에서 새로운 사이트를 만들어 리퀘스트를 제출할 수 있습니다.
