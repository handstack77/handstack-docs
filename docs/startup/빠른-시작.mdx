# 빠른 시작

이 문서는 Windows 10 이상 운영체제를 대상으로 합니다. macOS 및 Linux 환경에서는 메뉴얼로 필수 프로그램을 설치하여 HandStack을 실행할 수 있습니다.

## [필수] Windows 패키지 관리자 winget 설치

Microsoft에서 제공하는 Windows 패키지 관리자 winget 명령줄 도구는 Windows 11 및 최신 버전의 Windows 10에서 앱 설치 관리자로서 애플리케이션을 검색, 설치, 업그레이드, 제거 및 구성할 수 있습니다.

[앱 설치 관리자 설치하기](https://apps.microsoft.com/detail/9nblggh4nns1?rtc=1&hl=ko-kr&gl=KR#activetab=pivot:overviewtab)

## HandStack 개발 환경을 설정

개발자 PC에 HandStack을 설치하고 실행합니다.

https://git-scm.com/downloads 에서 Git을 다운로드하고 설치합니다.

최신 소스를 다운로드합니다.
```bash
git clone https://github.com/handstack77/handstack.git
```

다운로드한 소스로 이동하여 install.bat 배치 스크립트를 실행합니다.
```bash
cd handstack
install.bat
```

개발에 필요한 필수 설치 프로그램과 환경을 자동으로 구성합니다. 개발자 PC 사양과 네트워크 상태에 따라 10분 이상 소요될 수 있습니다.

설정이 완료되면 app 폴더로 이동하여 ack.exe 프로그램을 실행합니다.
```bash
cd handstack\1.WebHost\build\handstack\app
ack
```

## ack 실행 환경 설정

개발, 운영, 테스트 서버에 HandStack을 설치하고 실행합니다.

https://github.com/handstack77/handstack/releases/latest 에서 운영체제에 적합한 프로그램 압축 파일을 다운로드합니다.

* linux-x64.zip: 리눅스 운영체제
* osx-arm64.zip: ARM CPU 기반 macOS 운영체제
* osx-x64.zip: Intel CPU 기반 macOS 운영체제
* win-x64.zip: Windows 10 이상 운영체제

win-x64.zip 를 다운로드한 후 압축을 해제합니다.

압축을 해제한 폴더로 이동하여 install.bat 배치 스크립트를 실행합니다.
```bash
cd handstack
install.bat
```

실행에 필요한 필수 설치 프로그램과 환경을 자동으로 구성합니다. 서버 사양과 네트워크 상태에 따라 10분 이상 소요될 수 있습니다.

설정이 완료되면 app 폴더로 이동하여 ack.exe 프로그램을 실행합니다.
```bash
cd app
ack
```

## 웹 브라우저로 HandStack 실행 확인하기

ack 프로그램을 처음 실행하면 Administrator Email ID, Key를 확인할 수 있습니다. 이 정보는 checkup 모듈의 사용자 인증에 사용됩니다.

![](img/start-ack.png)

웹 브라우저를 열고 http://localhost:8000/checkup//account/signin.html 에 접속하여 HandStack을 실행합니다.
