# CLI 명령으로 ack 프로그램을 실행하는 배치 스크립트 만들기

## C:\bin 디렉토리 시스템 환경변수 등록

Windows 11에서 시스템 환경변수에 `C:\bin` 디렉토리를 추가하는 방법은 다음과 같습니다.

1. **시스템 속성 열기**
    - Windows + S 키를 눌러 검색창을 열고 "시스템 환경 변수 편집"를 입력합니다.
2. **환경 변수 창 열기**
    - 시스템 속성 창에서 “고급” 탭을 선택합니다. “환경 변수” 버튼을 클릭합니다.
3. **Path 변수 편집**
    - 시스템 변수 섹션에서 “Path” 변수를 찾아 선택한 후 “편집” 버튼을 클릭합니다.
4. **새 경로 추가**
    - “새로 만들기” 버튼을 클릭하고 C:\bin 경로를 입력합니다. “확인” 버튼을 눌러 변경 사항을 저장합니다.

---

## C:\bin 디렉토리에 ack-run.bat 만들기

1. **배치 파일 생성**
    ```bash
    notepad C:\bin\ack-run.bat
    ```

2. **배치 파일 내용 작성**
    ```bash
    @echo off
    cd %HANDSTACK_HOME%/app
    ack.exe
    ```

이제 `ack-run.bat` 파일을 실행하면 `ack` 프로그램이 실행됩니다.

```bash
ack-run
```
