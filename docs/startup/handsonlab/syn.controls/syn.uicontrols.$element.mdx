# syn.uicontrols.$element

사용자의 입력을 받을 수 없는 html 요소에 거래 요청과 응답을 위한 확장 기능을 부여합니다.

```html
<div id="divPanel" syn-datafield="Panel" syn-options="{content: 'html'}">innerHTML 테스트</div>
<span id="spnLabel" syn-datafield="Label" syn-options="{content: 'text'}">innerText 테스트</span>
<label id="txtApplicationID" syn-datafield="ApplicationID" syn-options="{content: 'value'}">value 테스트</label>
```
