# 지원 운영체제

> HandStack은 x64 아키텍처 기반의 Windows 11+, Ubuntu 22.04+, macOS 12+ 에서 개발하고 테스트합니다. 그러나 다음과 같이 운영체제에서도 사용이 가능합니다. 더 많은 자료는 공식 웹 사이트를 참고하세요.

## .NET 8 지원 운영체제

## Windows

|운영체제|버전|아키텍처|지원정보|
|---|---|---|---|
|[Windows 10 Client][Windows-client]|버전 1607+|x64, x86, Arm64|[Windows][Windows-lifecycle]|
|[Windows 11][Windows-client]|버전 22000+|x64, x86, Arm64|[Windows][Windows-lifecycle]|
|[Windows Server][Windows-Server]|2012+ (*)|x64, x86|[Windows Server][Windows-Server-lifecycle]|
|[Windows Server Core][Windows-Server]|2012+ (*)|x64, x86|[Windows Server][Windows-Server-lifecycle]|
|[Nano Server][Nano-Server]|버전 1809+|x64|[Windows Server][Windows-Server-lifecycle]|

## macOS

|운영체제|버전|아키텍처|
|---|---|---|
|[macOS][macOS]|12.0+|x64, Arm64|

M1, M2, M3와 같은 Arm64 CPU가 탑재된 macOS에서 .NET 8은 Rosetta 2 x64 에뮬레이터에서 지원됩니다.

[macOS]: https://support.apple.com/macos

## Node.js 20 지원 운영체제

Node.js 20 버전의 알려진 최소 운영체제 요구사항은 다음과 같습니다.

- **Windows**: Windows 7, Windows Server 2008 R2+
- **Linux**: Ubuntu 18+, RHEL / CentOS 8+, Debian 10+
- **macOS**: Big Sur(11.x)+
